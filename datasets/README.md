# Datasets

Data files are NOT stored in this repo (too large). Download them manually.

## Ottawa Bearing Dataset (used in Bouali et al. 2025)

- **URL:** https://data.mendeley.com/datasets/v43hmbwxpm/2
- **Citation:** Huang, H., & Baddour, N. (2019). Bearing Vibration Data under Time-varying Rotational Speed Conditions. Mendeley Data, V2.
- **Contents:** 60 recordings, each with vibration (Ch1) + encoder TTL (Ch2), 200 kHz, 10 seconds
- **Conditions:** 5 health states × 4 speed profiles × 3 trials
- **Download and extract to:** `datasets/ottawa-bearing/`

## CWRU Bearing Dataset

- **URL:** https://engineering.case.edu/bearingdatacenter
- **Contents:** Classic benchmark, constant speed, .mat files
- **Download and extract to:** `datasets/cwru/`
